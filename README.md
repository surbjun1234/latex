# LaTeX to PDF 생성기

## 프로젝트 개요

`latex` 프로젝트는 사용자가 웹 기반 인터페이스를 통해 LaTeX 코드를 입력하고, 이를 실시간으로 렌더링하여 PDF 문서로 변환 및 다운로드할 수 있도록 돕는 웹 애플리케이션입니다. 이 도구는 LaTeX 문서 작성의 편의성을 높이고, 복잡한 설치 과정 없이 웹 브라우저만으로 LaTeX 문서를 관리할 수 있게 합니다.

## 주요 기능

*   **LaTeX 코드 입력 및 미리보기**: 사용자 친화적인 에디터에서 LaTeX 코드를 작성하고, MathJax를 통해 실시간으로 렌더링된 결과를 확인할 수 있습니다.
*   **PDF 변환 및 다운로드**: 작성된 LaTeX 코드를 PDF 형식으로 변환하여 즉시 다운로드할 수 있습니다.
*   **클라이언트 측 처리**: 모든 변환 과정이 클라이언트 측에서 이루어져 서버 부하를 줄이고 사용자 개인 정보 보호를 강화합니다.

## 사용 기술

*   **HTML5, CSS3, JavaScript**: 웹 인터페이스 구현
*   **MathJax**: LaTeX 수식 렌더링
*   **`html2pdf.js`**: HTML 콘텐츠를 PDF로 변환

## 설치 및 실행 방법

이 프로젝트는 클라이언트 측에서만 동작하는 정적 웹 페이지이므로, 별도의 서버 설정 없이 웹 브라우저에서 `main.html` 파일을 열어 실행할 수 있습니다.

1.  **레포지토리 클론**: 
    ```bash
    git clone https://github.com/surbjun1234/latex.git
    cd latex
    ```

2.  **브라우저에서 열기**: 
    `main.html` 파일을 웹 브라우저로 드래그 앤 드롭하거나, 파일 경로를 직접 입력하여 엽니다.
    ```bash
    # 예시 (macOS)
    open main.html
    # 예시 (Windows)
    start main.html
    ```

## 파일 구조

```
latex/

```
